CI/CD GitHub Pages

Pipeline minimal avec GitHub Actions pour déployer automatiquement le site après chaque push.

DevOps

on: push
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      # Build si besoin
      - uses: actions/upload-pages-artifact@v3
        with:
          path: .
      - uses: actions/deploy-pages@v4

Durcissement SSH

Exemple de configuration sécurisée : désactiver root, forcer les clés, changer le port, fail2ban, etc.

Sécurité

PermitRootLogin no
PasswordAuthentication no
PubkeyAuthentication yes
Port 2222

Monitoring gratuit

Utilisation de Grafana Cloud (free) + node_exporter local pour visualiser CPU/RAM.
